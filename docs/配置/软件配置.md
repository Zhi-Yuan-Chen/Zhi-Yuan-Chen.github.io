## grammaly内嵌到word中

参考视频链接：[grammarly word插件安装全记录]( https://www.bilibili.com/video/BV1zT41177XP/?share_source=copy_web&vd_source=a54f81c2d9992921a2b3d9b9ebd487b5)

下载链接：[grammarly](https://www.grammarly.com/office-addin)

- 首先下载软件
- 下载完成后，对软件右键选择以管理员身份运行(!!!重要)
- 双击安装后，请按住`Ctrl+Shift`两个键的同时点击`Get Start` 按钮，grammarly会显示2条信息，点击 `Install for all users option`
- 可以自由选择下载位置
- 选择下载`for word`
- 安装即可，然后打开word，发现已经内嵌，点击`log in`进行登录
